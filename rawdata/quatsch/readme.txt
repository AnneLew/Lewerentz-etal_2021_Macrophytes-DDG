In diesem ZIP-Archiv finden Sie die angefragten Daten. 

Für jede Datenart (Flüsse/Abfluss, Meteorologie/Schnee, ...) finden Sie einen Ordner.
Innerhalb jedes Ordners finden Sie dann die gewünschten Daten in Form von einer oder 
mehrerer CSV-Dateien. 

Aus den Dateinamen können Sie den Inhalt ablesen. Die Dateien sind benannt nach dem Schema
<Messstellennummer>_<Beginndatum>_<Enddatum>_<Datenart>.csv
Die Messstellennummer ist eine eindeutige Identifikation der Messstelle. 
Beginn- und Enddatum beschreiben, welchen Zeitraum die Daten in der Datei abdecken. 
Die Datenart kann den Wert "ezw" (Einzelwerte) oder "tmw" (Tagesmittelwerte) annehmen.
